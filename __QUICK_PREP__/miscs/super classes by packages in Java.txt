Here's the superclass list for all java default packages:

Every class ultimately traces back to **`Object`** as the root superclass

---

**java.lang**
- `Object` → none (root)
- `String` → `Object`
- `StringBuilder` → `Object`
- `StringBuffer` → `Object`
- `Math` → `Object`
- `System` → `Object`
- `Thread` → `Object`
- `Throwable` → `Object`
- `Exception` → `Throwable`
- `RuntimeException` → `Exception`
- `Error` → `Throwable`
- `Number` → `Object`
- `Integer` → `Number`
- `Long` → `Number`
- `Double` → `Number`
- `Float` → `Number`
- `Byte` → `Number`
- `Short` → `Number`
- `Boolean` → `Object`
- `Character` → `Object`
- `Enum` → `Object`
- `Record` → `Object`

---
